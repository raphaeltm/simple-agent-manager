# CLI Tool

**Created**: 2026-02-23
**Priority**: Medium
**Classification**: `public-surface-change`, `cross-component-change`

## Context

Users have expressed interest in managing SAM from the terminal. A CLI would enable scripted workflows, CI/CD integration, and power-user agent orchestration. Referenced on the marketing site roadmap.

## Scope

Build a CLI tool (`sam`) that provides:

1. **Workspace management**: `sam workspace create`, `list`, `stop`, `restart`, `delete`
2. **Node management**: `sam node create`, `list`, `stop`, `delete`
3. **Agent interaction**: `sam agent start`, `sam agent chat` (stream ACP messages to terminal)
4. **Project management**: `sam project list`, `sam project create`
5. **Auth**: `sam login` (browser-based OAuth flow or API token)

## Technical Considerations

- TypeScript CLI using `commander` or `oclif`
- Reuse `@simple-agent-manager/shared` types
- Authenticate via API token or OAuth device flow
- Output formats: human-readable (default) + JSON (`--json`)
- Publish to npm as `@simple-agent-manager/cli`
