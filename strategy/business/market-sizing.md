# Market Sizing: SAM

**Last Updated**: 2026-03-11
**Update Trigger**: Annually, or when entering a new segment

## Market Definition

SAM operates at the intersection of two markets:
1. **Cloud Development Environments (CDEs)** — Infrastructure for remote/cloud dev
2. **AI Coding Agent Infrastructure** — Platforms that run AI coding agents

## SAM (Serviceable Available Market)

SAM's current architecture targets a specific subset:

| Filter | Reduction | Remaining |
|--------|----------|-----------|
| Developers using AI coding agents (not just copilots) | ~25% of AI tool users | ~4.7M |
| Want BYOC / self-hosted (not vendor-managed) | ~15-20% | ~700K-940K |
| Willing to use non-dominant platform (not Codespaces/Cursor) | ~30% | ~210K-280K |
| Average annual value per user | $300-600/year | — |

**SAM**: Approximately **$63M-$168M** in annual revenue potential.
