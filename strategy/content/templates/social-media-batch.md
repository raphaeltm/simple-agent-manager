# Template: Social Media Batch

Use this template to create a batch of platform-specific social posts from a single source (feature launch, blog post, milestone).

## Input
- **Source**: [link to the source artifact — blog post, changelog, spec]
- **Key Message**: [from messaging hierarchy]
- **CTA**: [what we want people to do]

## Output Format

```markdown
## Social Media: [Topic]

**Source**: [artifact link]
**Key Message**: [from messaging hierarchy]
**CTA**: [action]

### LinkedIn
[2-3 paragraphs, professional tone, ends with question or insight]

### Twitter/X Thread
1/ [Hook — most interesting point]
2/ [Context — why it matters]
3/ [Detail or example]
4/ [CTA with link]

### Twitter/X Single
[Under 280 chars, self-contained]

### Hacker News Title + Comment
**Title**: [Descriptive, not marketing-speak. "Show HN:" if applicable]
**Comment**: [Technical context, honest trade-offs, invite feedback. 3-5 paragraphs.]

### Reddit (r/[subreddit])
**Title**: [Community-appropriate]
**Post**: [Helpful framing. Focus on solving a problem, not promoting.]
```

## Quality Checklist
- [ ] Each platform version has unique formatting (not copy-paste)
- [ ] LinkedIn: professional, includes engagement question
- [ ] Twitter: punchy, thread has clear hook in tweet 1
- [ ] HN: technical, acknowledges trade-offs, no marketing language
- [ ] Reddit: community-first, provides value beyond promotion
- [ ] All versions use approved language from messaging guide
- [ ] CTA is appropriate for each platform
- [ ] Competitive claims are dated and sourced
