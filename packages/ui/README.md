# @simple-agent-manager/ui

Shared UI package for Simple Agent Manager.

## Scope

- Semantic design tokens
- Shared primitives
- Shared composed components
- Cross-surface UI patterns

## Consumers

- `apps/web`
- `packages/vm-agent/ui`

## Principles

1. Mobile-first by default
2. Accessibility and keyboard support are required
3. Use semantic tokens, not ad-hoc values
4. Keep components composable and agent-friendly

## Usage

```ts
import '@simple-agent-manager/ui/styles.css';
```
